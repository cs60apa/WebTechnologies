HTML Tag Examples and Use Cases

<a> Tag

The <a> tag is used to define hyperlinks. It can link to another page, a file, an email address, or any URL.

Example:

Visit Example.com

<abbr> Tag

The <abbr> tag is used to define an abbreviation or acronym. By hovering over the text, you can see the full description.

Example:

The WHO was founded in 1948.

<address> Tag

The <address> tag is used to define contact information for the author or owner of a document or article.

Example:

Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA

<article> Tag

The <article> tag specifies independent, self-contained content.

Example:

Article Title

This is an article. Articles can be blog posts, news stories, or other similar content.

<aside> Tag

The <aside> tag defines some content aside from the content it is placed in. It is often used for sidebars.

Example:

Related Articles

This is a sidebar with related articles.
