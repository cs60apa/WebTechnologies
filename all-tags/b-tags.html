HTML Tag Demonstrations

<base> Tag

The <base> tag sets the base URL for all relative links in the document. In this example, any relative URL will point to "https://example.com/".

Example: Image

<bdi> Tag

The <bdi> (Bidirectional Isolation) tag isolates text that may be different from the surrounding text direction. This is useful for user-generated content where the text direction is unknown.

Example: Username: إسم مستخدم

<bdo> Tag

The <bdo> (Bidirectional Override) tag explicitly sets the direction of the text it contains. You can use the dir attribute to specify "ltr" (left-to-right) or "rtl" (right-to-left).

Example: This text is right-to-left.

<blockquote> Tag

The <blockquote> tag is used to define a block of text that is quoted from another source. It usually renders with some indentation.

This is a blockquote from another source. It is typically indented and can include a citation URL.

<body> Tag

The <body> tag defines the main content of an HTML document. All the content that you see on the webpage is contained within the <body> tag.

<br> Tag

The <br> (line break) tag inserts a line break. It is useful for writing poems or addresses where the division of lines is significant.

<button> Tag

The <button> tag defines a clickable button. You can use it to submit forms or trigger JavaScript actions.
